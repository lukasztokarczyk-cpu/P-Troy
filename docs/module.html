Dashboard

Moduł

Ten moduł jest w pełni zaprojektowany w backendzie/frontendzie Next.js (folder backend/ i frontend/) — tutaj, w wersji statycznej pod GitHub Pages, pokazujemy jego miejsce na dashboardzie.

Pełna wersja tego modułu wymaga backendu (Node.js + PostgreSQL) i jest gotowa w katalogu backend/ tego repozytorium.

Statyczne demo obejmuje w pełni działające Harmonogram i Zadania.
